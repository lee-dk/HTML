속성 (Attribute)

속성이란? - 태그의 속성을 설정하는 값
속성은 적용하고 싶은 element의 여는 태그에 적어야한다 글로벌속성 - 모든 태그에 적용될 수 있는 속성
기본속성 - 그 태그에만 적용할 수 있는 속성

글로벌 속성 (Global Attribute)

  • style : CSS(스타일 시트)를 적용하는 속성
  • hidden : 설정한 태그를 숨긴다
  • id : 해당 태그를 유일하게 구분하기 위한 아이디 설정
  • class : 비슷한 태그끼리 그룹을 설정하는 용도로 사용
  • 다양한 이벤트 리스너들...
